# new-words.md

Word data file for Learn English Word app.
Push this file to your public GitHub repo, then set the raw URL in `AppConstants.wordsRemoteUrl`.

**Format per word:**
```
## <word>
- phonetic: <IPA>
- difficulty: <1=Easy | 2=Medium | 3=Hard>
- meaning: <Vietnamese meaning>
- definition: <English definition>
- usage: <grammar notes: prepositions, V2/V3, collocations, transitivity>
- example: <example sentence>
```
Fields `phonetic`, `definition`, `usage`, `example` are optional. `meaning` is required.

## accomplish
- phonetic: /əˈkʌmplɪʃ/
- difficulty: 2
- meaning: hoàn thành, đạt được
- definition: to succeed in doing or completing something
- usage: Ngoại động từ (V + O). Kết hợp: accomplish a goal / task / mission / objective. V2/V3: accomplished / accomplished. Thường dùng với "successfully", "fully".
- example: She accomplished her goal of learning three languages by the age of 25.

## abandon
- phonetic: /əˈbændən/
- difficulty: 2
- meaning: từ bỏ, bỏ rơi
- definition: to leave a place, person, or thing permanently with no intention of returning
- usage: Ngoại động từ (V + O). Kết hợp: abandon a project / hope / ship / idea. V2/V3: abandoned / abandoned. Giới từ: không cần giới từ. Thành ngữ: "abandon ship" (từ bỏ hoàn toàn).
- example: He abandoned his car on the side of the road and walked the rest of the way.

## inevitable

- usage: Tính từ. Kết hợp: inevitable consequence / result / outcome / conflict. Cấu trúc: it is inevitable that... | inevitable + N. Danh từ: inevitability. Phó từ: inevitably. Thường dùng như "the inevitable" (danh từ: điều không thể tránh).

- usage: Tính từ. Kết hợp: versatile tool / player / ingredient / material / actor. Danh từ: versatility. Phó từ: versatilely (ít dùng). Dùng cho cả người lẫn vật. Giới từ: không cần.
- example: This versatile kitchen appliance can be used for blending, chopping, and even cooking.
